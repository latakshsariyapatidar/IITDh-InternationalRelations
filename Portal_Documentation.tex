\section{Introduction}
Welcome to the comprehensive documentation for the International Relations Office (IRO) Portal of IIT Dharwad. This document serves as an exhaustive guide covering the public-facing website, the authenticated Student Portal, and the robust Administrator Panel. It is designed to help clients, administrators, and general users understand how each component functions, how they interconnect, and how modifications in the backend dynamically update the live public website.

\section{Public Website Overview}
The public-facing website acts as the primary source of information for prospective international students, current IITDh students, and global partner institutions. It retrieves all of its content dynamically from the backend database.

\subsection{Home Page}
The Home Page provides a welcoming overview of the IRO. 
\begin{itemize}
    \item \textbf{Announcements Marquee:} A scrolling ticker at the very top displaying real-time updates and public announcements. Clicking an announcement reveals more details and clickable links.
    \item \textbf{Welcome Message \& Statistics:} Introduces the office and displays dynamic statistics (e.g., number of active MOUs, international partners).
    \item \textbf{Testimonials Carousel:} Highlights experiences of past exchange students.
\end{itemize}

\section{Student Portal}
The Student Portal is designed primarily for IIT Dharwad students looking to apply for outbound exchange programs, research internships, and other international opportunities.

\subsection{Authentication}
Students log in to the portal securely using their institutional Google accounts (OAuth). This seamless single sign-on (SSO) ensures that only verified students can access the application portal.

\subsection{Outbound Applications Submission}
Once authenticated, students can submit applications for various outbound programs (e.g., Semester Exchange, Research Internship, Summer Program, Dual Degree). They must provide academic details, their intended semester, motivation (Statement of Purpose), and upload necessary PDF documents (transcripts, recommendation letters).

\section{Admin Panel (Content Management System)}
The Admin Panel is the secure backend control center. Every CRUD (Create, Read, Update, Delete) module directly manipulates the PostgreSQL database and immediately updates the content on the live public website.

\subsection{Admin Authentication \& Dashboard}
Admins log in using their authorized credentials. The Dashboard provides a high-level statistical overview of portal activity (Pending Inbound/Outbound applications, Total Partners, Active MOUs, etc.).

\section{Summary}
The IRO Portal is engineered to be a living, breathing application. The Admin Panel is securely built to manage almost every aspect of the public website dynamically. By ensuring that all records, applications, and announcements are kept up-to-date in the Admin Panel, the administrative team guarantees that the public website and student portal remain accurate, engaging, and highly functional for its global audience.
